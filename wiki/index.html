Welcome to the scratch-bookmarklets wiki. This replaces the built in github wiki for any repository and contains information on how to contribute,
how to use these bookmarklets, and more.
Scratch-bookmarklets is an open source project where bookmarklets are made and then put up on the internet.
To contribute, see our [contributing guidelines] and/or our [contributing page]. All comments and commits should follow our [Code Of Conduct].
